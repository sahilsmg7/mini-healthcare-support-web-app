# Arogya Seva - Mini Healthcare Support Web App

## 🌍 The NGO Use-Case
**Arogya Seva** is a conceptual non-profit organization designed to democratize healthcare in India. The application bridges the critical geographical gap between urban medical specialists and rural patients. 

While top-tier doctors are concentrated in cities, rural areas suffer from a lack of immediate, high-quality medical triage. This platform acts as a digital bridge—allowing specialist doctors from across the nation to register as volunteers and providing rural patients with instant, automated tele-triage guidance before they book a remote consultation.

## 💻 Tech Stack
- **Frontend Framework:** React (Vite)
- **Styling:** Tailwind CSS (configured for a modern Blues/Whites/Teals UI)
- **Icons:** Lucide-React
- **Animations:** Custom Tailwind CSS utilities for smooth transitions and fading carousels

## 🤖 AI Features Implemented
1. **AI Smart Symptom Analyzer (Triage):**
   A dedicated section on the application where patients can type their symptoms in plain English. An active Natural Language mechanism processes these keywords to determine the **Urgency Level**, recommends the specific **Medical Specialist** required (e.g., Cardiologist vs. General Physician), and provides instant **AI Advice**.
   
2. **AI Triage Assistant (Chatbot):**
   A floating widget that actively handles automated FAQs. It simulates an AI conversation to instantly guide patients on how to book consultations, verify free services, or assist doctors on how to volunteer.

## 🚀 How to Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
